# Design

## Architecture

- Cli: Terminal access
- Service: Quick access service to the models
- Integration: generated by OpenAPI doc
- Syncer: Sync local storage with remote api server
- Persistent: Data persistence with boltdb(k-v) and go-memdb(indexing)


```
--------------------
     cli | openapi
--------------------
         | syncer
--------------------
      service 
--------------------
         | integration
--------------------
    persistence (boltdb + go-memdb)
--------------------
```
